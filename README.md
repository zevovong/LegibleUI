#Legible UI

Legible UI 是一个轻量级、语义化的前端 UI 框架。

##浏览器支持
- Chorme 21+
- Opera 12.1+
- Firefox 22+
- Safari 6.1+
- IE 10+

##项目文件
###css/

- legible.core.css

    基础的 CSS UI 样式，不需要加载 JS

- legible.widget.css

    包含一些扩展，需要加载 JS，包含 legible.core

###js/

- legible.widget.js

    组件对应的 JS 库，需要 **jQuery** 或者 **Zepto** 的支持。
