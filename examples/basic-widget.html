Button 按钮

我们可以通过添加 .button 来为元素设置按钮样式。

Button

参考代码：

<button class="button">Button</button>
<input class="button" type="button" value="Button">
<a class="button" href="#">Button</a>

但是，我们只建议在一下几种元素上使用按钮样式

  • <button>
  • <input type="button | submit | reset">
  • <a href="# | http://...">

按钮的颜色

按钮可以有几种样式，我们为你设置了一些基本样式。你可以你的需求自己修改。

Circle Button 圆形按钮

我们可以通过添加 .button.circle 来为元素设置按钮样式。

Example 示例

<button class="circle button">Button</button>

Naked Button 无样式按钮

我们可以通过添加 .naked 来为元素设置按钮样式。

Example 示例

<button class="naked">Button</button>
<button class="naked button">Button</button>

文本框（Textbox）

我们可以通过添加 .textbox 来为表单输入框设置样式。

参考代码：

<input class="textbox" placeholder="textbox" />
<textarea class="textbox" placeholder="textbox"></textarea>

自适应输入框

在 textarea 上使用 . 来使得 textarea 自适应
